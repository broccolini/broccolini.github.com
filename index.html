Oh Hai!

Hello, I'm Diana Mounter.

I design user experiences and help teams build them. My approach is influenced by human-centric design, agile development, the people I work with, and the lessons I learn from each project.

Design

I sketch, illustrate, and craft designs in Photoshop and Illustrator for websites and printed things.

  • GitHub "black swan" posters

    I wanted to stretch my Illustrator skills so I created myself a project, which was to design a poster for GitHub as if it were a film in the style of the Black Swan movie posters using the GitHub Octocat as the main visual feature.

    I didn't have an EPS file of the original Octocat so I had to trace the different versions from images I found on the web - apologies for any inaccuracies!

  • Stick Party

    The Stick Party site was for advertising a web app that a friend of mine was building. He wanted it to be a fun design and liked the hand-drawn sketched look.

    I created "Mr Stick" which I think help achieved the fun feel.

  • Eco2labs

    This website design was for a startup I was working with that was building a database and web applications on the sustainability of products. The aims of the site were to encourage people to sign up and get involved, and provide basic information to potential business partners and clients.

    There was no branding established and the exact direction for the business was still being developed, so we agreed to go with a fairly simple site design that would reflect the higher level goal of helping people live more sustainably.

  • Stories magazine

    An online and printed magazine for people working on the web in Local Government. I designed and customised the Wordpress site, and managed the production of the printed version for the first issue.

    I'm no longer involved in producing the magazine or the website though it's close to the original version, apart from the bits that have now broken!

  • I am also pretty good at drawing Poodles

Projects

I research and design user experiences, and develop concepts into UI specifications.

  • Virtual Learning Environment

    I was the producer and developed the user experience strategy for an online learning tool. We were asked to develop a prototype though the first version was used for a real course by students in the Asia Pacific.

  • Urban Sketch Lab

    I was the producer and experience architect for developing an interactive installation and accompanying event website and blog. This project taught me a lot about designing interactions for use during and after events.

  • Designing broccolini.net

    Being your own client is hard. To get past staring at a blank screen I took a strategic approach like I usually would on projects at work. I allowed myself just one weekend to design and build it so it is essentially the MVP version of my own website.

  • More projects I'm proud of: Bouncr; ACF GreenHome; UNSW Sustainability; LGSA Cultural Awards 2009.
