I wanted to stretch my Illustrator skills so I created myself a project, which was to design a poster for GitHub as if it were a film in the style of the Black Swan movie posters using the GitHub Octocat as the main visual feature.

This was before GitHub created their new about page and made the EPS file available for all to use, so I had to trace the Octocat from scratch - apologies for any inaccuracies.
